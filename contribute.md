## Guidelines for Submissions

When submitting a new tool, please ensure it meets the following criteria:

- **Open Source**: The tool must be open-source with a publicly accessible repository
- **Trust and Safety Related**: The tool should be related to AI safety in some way
- **Active**: Preferably, the tool should be maintained or have been updated recently
- **Description**: Provide a clear, concise description of what the tool does
